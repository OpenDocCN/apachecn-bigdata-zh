## 下载

### Docker

```
docker pull apachecn0/apachecn-bigdata-zh
docker run -tid -p <port>:80 apachecn0/apachecn-bigdata-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-bigdata-zh
apachecn-bigdata-zh <port>
# 访问 http://localhost:{port} 查看文档
```
