## 下载

### Docker

```
docker pull apachecn0/flygon-bigdata-zh
docker run -tid -p <port>:80 apachecn0/flygon-bigdata-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-bigdata-zh
flygon-bigdata-zh <port>
# 访问 http://localhost:{port} 查看文档
```
